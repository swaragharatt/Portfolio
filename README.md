
# Portfolio 

**Swara Gharat Portfolio** is a responsive dark-themed portfolio website built using HTML, CSS, and JavaScript. It highlights Swara’s front-end skills, project accomplishments, and professional goals with a clean, elegant interface.

---

## Technologies Used

| Language | Styling        | Icons        | Fonts |
| -------- | -------------- | ------------ | ----- |
| HTML     | CSS (internal) | Font Awesome | Inter |

---

## Features

* Dark-themed modern UI with smooth animations
* Personal avatar integration and hero introduction
* Interactive navigation bar with hover underline effect
* Project listings and skill highlights
* Footer with resume download and contact details

---

## Project Structure

```
swara-portfolio/
├── index.html           # Main HTML file with embedded CSS
├── swara-profile.jpg    # Profile picture used in the hero section
```

---

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/swaragharatt/swara-portfolio.git
cd swara-portfolio
```

Or simply download and open `index.html` in your browser.

---

## Usage

Open `index.html` in any modern browser. The site will display:

* A profile image and greeting with Swara’s name and role
* A section outlining her career objective
* A list of key technical and personal projects
* A breakdown of technical and soft skills
* Contact information with external resume link
